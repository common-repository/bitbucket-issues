=== Relative Menu Item ===

Get issues for bitbucket repositories


== Description ==

Aggregate issues from all repositories owned by a bitbucket.org user and display as list.

Shortcode-based at the moment.

Co-Developers appreciated.

== Important Notes ==

Not tested with multisite installations.


== Installation ==

Check out the [official installation guide](http://codex.wordpress.org/Managing_Plugins). It's totally more awesome than anything I'd be bothered to write here. Plus, wordpress codex is your friend. (Mostly).

This plugin has its [own wiki](https://bitbucket.org/perchten/wordpress-bitbucket-issues-plugin/wiki). Any info on it will be there.
